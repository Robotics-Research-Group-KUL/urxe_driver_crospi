# urXe_driver_crospi Package

This package contains a driver for controlling the UR CB and e series using crospi_core, based on template_driver_crospi package which uses ROS2 plugins.

## Installation

Place under external folder the [ur_rtde](https://gitlab.com/sdurobotics/ur_rtde) library.

## Documentation

For documentation and tutorials, checkout our [crospi website](https://crospi-website-907f83.pages.gitlab.kuleuven.be/).
